# 🌸 Единый деканат

Веб-приложение для управления учебным процессом в вузе.  
Студенты могут смотреть оценки и долги, следить за дедлайнами, преподаватели — выставлять оценки и устанавливать дедлайны, деканат — управлять долгами и формировать отчёты.

---

## О проекте

| Роль | Возможности |
|:--|:--|
| **Администратор** | Управление пользователями, ролями, факультетами |
| **Деканат** | Просмотр всех студентов, управление долгами, отчёты, приказы |
| **Преподаватель** | Ввод оценок, просмотр своих групп, пересдачи |
| **Студент** | Просмотр своих оценок, долгов, уведомлений |

В проекте использованы 2 локальных представления: Успеваемость и Дедлайны: преподаватели, студенты, уч процесс декомпозированы для простоты

### 2. Создать виртуальное окружение

```bash

**Windows (PowerShell):**
```bash

```

**Windows (CMD):**
```bash

```

**Mac/Linux:**
```bash
source venv/bin/activate

```bash

```

```bash

```

## 🗄️ Работа с базой данных

База данных находится в файле `instance/university.db`. Это SQLite-файл, который можно открыть с помощью:

### DB Browser for SQLite (рекомендуется) в VS Code нет работающих(поддерживаемых) расширений с этой бд, не пытайтесь

**Что это такое?**  
DB Browser for SQLite — это бесплатная программа с графическим интерфейсом, которая позволяет просматривать и редактировать SQLite-базы данных без написания SQL-запросов.

**Где скачать?**  
👉 [https://sqlitebrowser.org/dl/](https://sqlitebrowser.org/dl/)

**Как открыть базу данных:**
1. Установить DB Browser for SQLite
2. Открыть программу
3. Нажать **"Open Database"** фотка в тг
4. Выбрать файл `instance/university.db`
5. На вкладке **"Browse Data"** выбрать таблицу для просмотра

**Что можно делать:**
- Просматривать все таблицы и данные
- Редактировать ячейки
- Выполнять SQL-запросы на вкладке **"Execute SQL"**
- Экспортировать данные в CSV, JSON, Excel
